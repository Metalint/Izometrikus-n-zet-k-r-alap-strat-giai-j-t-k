A játék futtatásának Lépései

1. lépés: Letöltjük a Godot motort a hivatalos weboldalukról.

2. lépés: Elhelyezzük kívánt helyre a Godotott kívánt helyre a gépen a zip fáljból.

3. lépés: indítsuk el a Godot_v4.5-stable_mono_win64.exe fáljt ami elő hozza a Godot motor felületétt.

4.lépés: A játék importálása a Godotba. A legegyszerűbb, ha a projekt mappáját a dokumentumok közé rakjuk és onnan importáljuk a Godotba.

5. Lépés: Futtatjuk a projektet. alternatívan a kódót meg lehet nézni kódoló programmal (én a Visual Studio Code alkalmazást használtam.).
